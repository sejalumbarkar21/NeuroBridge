# NeuroBridge
# 🧠 NeuroBridge – Alzheimer’s Support System

## 📌 Overview

**NeuroBridge** is a JavaFX-based application designed to assist Alzheimer’s patients and their families.
It focuses on improving memory retention, daily routine management, and emotional connection through interactive modules.

---

## 🎯 Key Features

### 🧩 Memory Games

* **Level 1 – Knowing Myself** (video-based memory recall)
* **Level 2 – Faces of Love** (recognizing close people)
* **Level 3 – Quiz Module** (dynamic MCQ-based memory testing)
* **Level 4 – Emotional Letter** (supportive message UI)

---

### 👨‍👩‍👧 Family Integration

* Family members can upload:

  * Images
  * Videos
  * Memory-related content
* Patients can access personalized data using a **unique Patient ID**

---

### 📅 Daily Routine Planner

* Family can assign tasks (Morning / Day / Night)
* Patients receive structured daily schedules

---

### 🏥 Hospital & Appointment Module

* View Alzheimer’s hospitals
* Check doctor details
* Book appointments

---

### 🧘 Meditation Module

* Focus, Calm, Relax modes
* Audio playback with timer

---

### 🔐 Authentication System

* Separate login for:

  * Patients
  * Family members
* Firebase Authentication integration

---

## 🛠️ Tech Stack

* **Frontend:** JavaFX
* **Backend:** Firebase (Firestore + Storage + Auth)
* **Language:** Java
* **API Integration:** Gemini API (for dynamic quiz generation)

---

## 📂 Project Structure

```
src/
 ├── main/java/com/neurobridge/   # Core Java classes
 ├── resources/Assets/            # Images, audio, UI assets
```

---

## 🚀 How to Run

1. Clone the repository

   ```
   git clone https://github.com/your-username/NeuroBridge.git
   ```

2. Open in IntelliJ / Eclipse

3. Configure Firebase:

   * Add your own Firebase config file (DO NOT upload it publicly)

4. Run:

   ```
   Main.java
   ```

---

## 🔒 Security Note

Sensitive files like Firebase credentials are **not included** in this repository.
Please configure your own credentials locally.
